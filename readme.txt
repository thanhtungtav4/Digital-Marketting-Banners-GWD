:::::::::::::::::::> BANNER CUSTOMIZATION INSTRUCTIONS <:::::::::::::::::::::::::::::::::::::

Navigate to the folder containing the HTML5 banner files that you purchased. Open folder "main" including Main_PSD and Main_HTML banner file.

Banner customization only with Photoshop and google web design. 


:::::::::::::::::::> HOW TO ADD YOUR TEXT<:::::::::::::::::::::::::::::::::::::

Each banner folder has a html file. You open it up in notepad.

You find text (use crt +f ) and replace your text.

:::::::::::::::::::> HOW TO ADD YOUR LOGO AND IMAGE <:::::::::::::::::::::::::::::::::::::

You can directly replace the images file (logo.png) found in the "assets" folder of the banner you are customizing. 

Banner best if your image size similar to the size of existing images in the "assets" folder

:::::::::::::::::::> HOW TO ADD YOUR IMAGE <:::::::::::::::::::::::::::::::::::::

You can directly replace the images file (mainpic.png) found in the "assets" folder of the banner you are customizing. 

Banner best if your image size similar to the size of existing images in the folder images

:::::::::::::::::::> EDIT WEBSITE LINK <:::::::::::::::::::::::::::::::::::::

Each banner folder has a html file. You open it up in notepad.

You find links http://banner.nanothemes.co/ and replace your link.

Then Saved. 

:::::::::::::::::::> EDIT FOR ADWORDS <:::::::::::::::::::::::::::::::::::::

We have added the google's requirement to file. After customization,  you only need to zip your file.

Note: optimize images, for a total capacity banner after zip < 150kb


:::::::::::::::::::> HOW TO PLACE YOUR BANNER ON A WEBPAGE <:::::::::::::::::::::::::::::::::::::

Upload your published banner files to a webserver.

The banners can be placed on a webpage with the "iframe" HTML markup. See sample code below:

<iframe src="300x600/300x600.html" frameborder="0" scrolling="no" width="300" height="600" style="border:solid 1px #999;"></iframe>
